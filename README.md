# Classy Flappy Bird (Web Edition)

A fully playable, responsive browser-based arcade game built with a Python Flask backend and an HTML5 Canvas frontend, featuring custom background themes and dynamic difficulty scaling.

I am a student who loves to code and build things, I have taken on many building projects that i am yet to post, these include rockets, and robotic arms. This project was coded in python, on VS code, this was a game that I used to play but was outdated, I wanted to re create one by myself using the knowlege of my coding skills. The hardest part was most definietly the tkinter parts as I was still new to that.

About the game
The game is a simple re creation of Flappy bird and coded on VS code utilizing python.
